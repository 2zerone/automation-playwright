# VIOLA - 새로운 테스트 플랫폼

VIOLA는 향상된 사용자 경험과 직관적인 인터페이스를 제공하는 새로운 테스트 플랫폼입니다.

## 특징

- 🎨 **연한 연두색 테마**: 시각적으로 편안한 연한 연두색 컬러 팔레트
- 🤖 **AI 기반 자동화**: 지능형 테스트 자동화 기능
- 📊 **실시간 모니터링**: 테스트 진행 상황 실시간 추적
- 🔒 **보안 강화**: 최신 보안 기술 적용

## 파일 구조

```
VIOLA/
├── viola-main.html          # VIOLA 메인 화면
├── README.md                # 이 파일
└── (추가 파일들이 여기에 생성될 예정)
```

## 사용법

1. Electron 앱을 실행합니다
2. 제품 선택 화면에서 "VIOLA"를 선택합니다
3. VIOLA 메인 화면이 로드됩니다
