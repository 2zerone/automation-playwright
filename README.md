# Trombone UI Test Automation

Trombone 시스템의 UI 자동화 테스트 프로젝트입니다.

## 🚀 빠른 시작

### 1. 의존성 설치
```bash
npm install
```

### 2. 애플리케이션 실행
```bash
npm start
```

### 3. 테스트 실행
```bash
npm test
```

## 📁 프로젝트 구조

자세한 폴더 구조는 [docs/FOLDER_STRUCTURE.md](docs/FOLDER_STRUCTURE.md)를 참조하세요.

### 주요 폴더
- **`src/`** - 소스 코드
- **`reports/`** - 보고서 및 문서
- **`test-files/`** - 테스트 관련 파일들
- **`config/`** - 설정 파일들
- **`tests/`** - Playwright 테스트

## 🔧 주요 기능

- **시나리오 기반 테스트**: 21개 시나리오 지원
- **자동화된 UI 테스트**: Playwright 기반
- **리포트 생성**: HTML, 마크다운, 커스텀 리포트
- **녹화 기능**: 테스트 실행 과정 녹화
- **Electron 앱**: 데스크톱 애플리케이션
