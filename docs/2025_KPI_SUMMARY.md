## KPI #1: 테스트 자동화 스크립트 작성 및 데스크톱 애플리케이션 제작
: 4개 제품(TROMBONE, VIOLA, CMP, CONTRABASS)의 시나리오 테스트 자동화를 통합하여 수행할 수 있는 데스크톱 애플리케이션 제작

### 목표 및 기대효과
> 시나리오 테스트 및 회귀 테스트를 자동화하여 검증 일관성 유지 및 검증 시간 단축
> GUI 기반 데스크톱 애플리케이션을 통해 손쉽게 시나리오 테스트를 수행할 수 있는 환경 구축
> 4개 제품에 대한 통합 테스트 프레임워크 구축 및 공통 라이브러리 개발로 코드 재사용률 85% 달성
> 수동 테스트 대비 실행 시간 80% 단축으로 연간 약 800시간 절감

### 산출물
> Electron 기반 테스트 실행 데스크톱 애플리케이션 (4개 제품별 구축)
> 71개 시나리오 파일, 349개 자동화 테스트 케이스 (총 88,584줄)
> 31개 공통 라이브러리 클래스 (LoginManager, ReportGenerator, ScenarioRunner 등 5,491줄, 재사용률 85%)
> HTML 시각화 리포트, 자동 스크린샷, 비디오 녹화 기능
> 프로젝트 구조 문서 및 README 작성

### 평가방식
- **정량 평가:** 프레임워크 구축 (4/4 제품, 100%), 코드 작성량 (94,075줄, 118%), 테스트 케이스 (349개, 116%), 재사용률 (85%, 121%)
- **측정 기준:** 코드 라인 수(`wc -l`), 테스트 케이스 수(`test()` 함수), E2E 완성도(전체 플로우 통과)

## KPI #2: 화면 녹화(Codegen)를 통한 테스트 케이스 자동 생성 도구 개발
: Playwright Codegen과 Google Sheets를 연동한 테스트 케이스 자동 생성 및 관리 시스템 구축

### 목표 및 기대효과
> 화면 녹화를 통한 테스트 케이스 자동 생성으로 작성 시간 85% 이상 단축 (2-3시간 → 10-15분)
> 비개발자(QA, 기획자)도 테스트 케이스를 생성할 수 있는 노코드/로우코드 환경 제공
> Google Sheets 기반 중앙 집중식 테스트 케이스 관리로 팀 단위 협업 기반 마련
> 동일 시간에 10배 많은 테스트 케이스 생성 가능하여 커버리지 확대 및 연간 약 400시간 절감

### 산출물
> Codegen 자동화 스크립트 6종 (codegen-auto-sheets, codegen-to-sheets, sheets-to-playwright-direct 등 총 2,742줄)
> Google Sheets API 연동 시스템 (Service Account 인증, 양방향 자동 변환)
> REST API 서버 5개 엔드포인트 (데이터 조회/추가/삭제/케이스 목록)
> 사용 가이드 문서 (README 186줄, API 명세, 워크플로우 다이어그램)
> 자동 생성 테스트 케이스 5개 (tests/generated)

### 평가방식
- **정량 평가:** 도구 개발 (6/4개, 150%), 코드 규모 (2,742줄, 137%), 시간 절감 (85-90%, 106%), API 엔드포인트 (5/3개, 167%)
- **측정 기준:** 작성 시간 비교 (수동 120분 vs 자동 12.5분 = 90% 절감)
