# Playwright MCP 로그인 후 특정 화면 이동 프롬프트

## 기본 사용법

다음 프롬프트를 사용하여 Playwright MCP로 로그인 후 특정 화면으로 이동할 수 있습니다:

```
Playwright MCP를 사용하여 다음 작업을 수행해주세요:

1. 브라우저를 열고 [웹사이트 URL]에 접속
2. 로그인 페이지에서 다음 정보로 로그인:
   - 사용자명: [사용자명]
   - 비밀번호: [비밀번호]
3. 로그인 성공 후 [목적지 화면/메뉴]로 이동
4. 해당 화면이 정상적으로 로드되었는지 확인
5. 현재 화면의 스크린샷을 촬영

각 단계별로 진행 상황을 알려주고, 오류가 발생하면 상세한 오류 메시지를 제공해주세요.
```

## 구체적인 예시

### 예시 1: Trombone 시스템 로그인 후 티켓 관리 화면 이동

```
Playwright MCP를 사용하여 다음 작업을 수행해주세요:

1. 브라우저를 열고 http://localhost:3000에 접속
2. 로그인 페이지에서 다음 정보로 로그인:
   - 사용자명: admin
   - 비밀번호: password123
3. 로그인 성공 후 "티켓 관리" 메뉴로 이동
4. 티켓 관리 화면이 정상적으로 로드되었는지 확인
5. 현재 화면의 스크린샷을 촬영

각 단계별로 진행 상황을 알려주고, 오류가 발생하면 상세한 오류 메시지를 제공해주세요.
```

### 예시 2: GitLab 로그인 후 프로젝트 페이지 이동

```
Playwright MCP를 사용하여 다음 작업을 수행해주세요:

1. 브라우저를 열고 https://gitlab.com에 접속
2. 로그인 페이지에서 다음 정보로 로그인:
   - 사용자명: myusername
   - 비밀번호: mypassword
3. 로그인 성공 후 "Projects" 메뉴로 이동
4. 프로젝트 목록이 정상적으로 표시되는지 확인
5. 현재 화면의 스크린샷을 촬영

각 단계별로 진행 상황을 알려주고, 오류가 발생하면 상세한 오류 메시지를 제공해주세요.
```

### 예시 3: 특정 폼 페이지로 직접 이동

```
Playwright MCP를 사용하여 다음 작업을 수행해주세요:

1. 브라우저를 열고 http://localhost:3000에 접속
2. 로그인 페이지에서 다음 정보로 로그인:
   - 사용자명: testuser
   - 비밀번호: testpass

4. 사용자 등록 폼이 정상적으로 표시되는지 확인
5. 폼의 각 입력 필드가 올바르게 렌더링되었는지 확인
6. 현재 화면의 스크린샷을 촬영

각 단계별로 진행 상황을 알려주고, 오류가 발생하면 상세한 오류 메시지를 제공해주세요.
```

## 고급 옵션

### 대기 시간 조정
```
로그인 후 3초 대기한 다음 메뉴로 이동해주세요.
```

### 특정 요소 확인
```
로그인 성공 후 "대시보드" 텍스트가 화면에 표시되는지 확인해주세요.
```

### 다중 단계 이동
```
로그인 후 다음 순서로 이동해주세요:
1. "관리" 메뉴 클릭
2. "사용자 관리" 하위 메뉴 클릭
3. "사용자 등록" 버튼 클릭
```

### 오류 처리
```
각 단계에서 오류가 발생하면:
1. 오류 메시지를 상세히 기록
2. 현재 화면의 스크린샷 촬영
3. 다음 단계로 진행할지 중단할지 결정
```

## 주의사항

1. **보안**: 실제 비밀번호는 프롬프트에 직접 포함하지 말고, 환경변수나 별도 파일을 사용하세요.
2. **네트워크**: 로컬 개발 환경과 실제 서버 환경의 URL을 구분하여 사용하세요.
3. **대기 시간**: 네트워크 상황에 따라 적절한 대기 시간을 설정하세요.
4. **요소 선택**: 동적으로 생성되는 요소의 경우 적절한 대기 시간을 설정하세요.

## 활용 예시

이 프롬프트는 다음과 같은 상황에서 유용합니다:

- **테스트 자동화**: 로그인 후 특정 화면으로 이동하여 테스트 수행
- **데모 준비**: 특정 화면을 미리 준비하여 데모 진행
- **디버깅**: 로그인 후 문제가 있는 화면으로 이동하여 상태 확인
- **스크린샷 촬영**: 특정 화면의 스크린샷을 자동으로 촬영
- **UI 검증**: 로그인 후 UI 요소들이 올바르게 표시되는지 확인
