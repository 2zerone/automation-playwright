# CMP - 컴포넌트 테스트 플랫폼

CMP는 모듈화된 컴포넌트 테스트 플랫폼입니다.

## 특징

- 🎨 **보라색 테마**: 창의적이고 혁신적인 보라색 컬러 팔레트
- 🧩 **모듈화**: 컴포넌트 기반의 모듈화된 테스트 구조
- 🔄 **재사용성**: 높은 재사용성을 위한 컴포넌트 설계
- ⚡ **빠른 개발**: 컴포넌트 조합을 통한 빠른 테스트 개발

## 파일 구조

```
CMP/
├── cmp-main.html          # CMP 메인 화면
├── README.md              # 이 파일
└── (추가 파일들이 여기에 생성될 예정)
```

## 사용법

1. Electron 앱을 실행합니다
2. 제품 선택 화면에서 "CMP"를 선택합니다
3. CMP 메인 화면이 로드됩니다

## 개발 예정 기능

- [ ] 컴포넌트 라이브러리 관리
- [ ] 모듈 조합 도구
- [ ] 재사용 가능한 테스트 템플릿
- [ ] 컴포넌트 의존성 관리
- [ ] 자동화된 컴포넌트 테스트
